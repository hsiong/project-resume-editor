# 项目经验

## XX 科技有限公司 (2020.01-至今)

### 企业级通用管理系统

**角色**：后端开发工程师

**技术栈**：Java, Spring Boot, MySQL, Redis, Vue.js

**项目描述**：为中小企业提供通用的后台管理功能，包括用户管理、权限控制、报表统计等基础模块，帮助客户提升办公效率。

**核心职责与产出**：

- **权限模块开发**：
  - 基于 RBAC 模型设计权限管理系统，实现精确到按钮级别的权限控制。
  - 优化查询逻辑，将权限校验耗时降低了 50%。
- **报表统计功能**：
  - 采用定时任务拉取数据并计算，将报表生成时间从分钟级降低至秒级。
- **系统优化**：
  - 引入 Redis 缓存高频访问数据，提升系统整体吞吐量。
  - 完成系统容器化改造，实现快速部署和扩容。

**角色**：Java 研发工程师

**技术栈**：Java, Spring Cloud, MyBatis, RabbitMQ

**项目描述**：参与公司自研 B2C 电商平台的核心交易链路研发，支撑日常促销活动和订单流转。

**核心职责与产出**：

- **消息队列应用**：
  - 引入 RabbitMQ 实现订单异步下单、库存扣减与消息通知，有效削峰填谷。
