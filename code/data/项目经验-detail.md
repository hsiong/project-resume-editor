## 示例项目一：企业级中后台管理平台

**项目描述：**  
该项目为公司内部核心的业务管理中枢，覆盖了订单、库存、财务等多个模块，每日活跃人数过万，系统面临高并发和复杂表单交互的挑战。

**职责与技术栈：**  
*   **技术栈：** Vue 3 + TypeScript + Vite + Element Plus + Pinia
*   作为前端核心骨干，负责整体架构搭建、通用组件库开发及核心模块的业务编写。
*   主导表单引擎的开发，将原本复杂的硬编码表单抽象为 JSON Schema 配置，开发效率提升 40%。
*   优化了首屏加载速度，引入路由懒加载和静态资源 CDN 加速，将 LCP 降低至 1.2s。

## 示例项目二：跨端移动电商应用

**项目描述：**  
面向终端用户的电商 App，需要在 iOS、Android 和 H5 三端保持高度一致的用户体验。

**职责与技术栈：**  
*   **技术栈：** React Native + Redux + TypeScript
*   负责首页、商品详情页及购物车模块的开发。
*   通过长列表渲染优化（虚拟列表）解决了低端机型滚动卡顿的问题，提升了流畅度。
*   与原生端配合，编写部分桥接代码（Bridge），实现扫码、蓝牙打印等定制功能。
